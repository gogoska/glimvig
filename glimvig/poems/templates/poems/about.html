О проекте

Glimvig — это литературная социальная сеть, где авторы могут публиковать свои произведения и связывать их с другими работами через идеи, персонажей или сюжеты.

Концепция «Цепочка поэзий»

Каждое произведение может быть связано с другими — продолжением, вдохновением, общим персонажем. Эти связи создают единое смысловое поле, которое можно исследовать визуально через интерактивный граф.

Возможности
  • Публикация стихов, рассказов, отрывков
  • Рейтинг и обсуждения
  • Создание связей между произведениями
  • Визуализация графа связей
  • Поиск по тегам и категориям
Технологии
  • Django + PostgreSQL
  • Bootstrap 5, HTMX, Alpine.js
  • Force‑Graph для визуализации
  • Docker, Git
